# fegtrack-languages

Find the language you would like to translate and replace what is in the quotes with the translation.

## Example

English (en-US)\
ADD_NEW_WALLET: "Add New Wallet"

Translated to Russian (ru-RU)\
ADD_NEW_WALLET: "Добавить кошелек"
